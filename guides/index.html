使用教程

从启动 ComfyUI 到选择模型架构，快速上手本站生图工具

1. 启动 ComfyUI

启动时需要加 --enable-cors-header 参数以允许网页跨域访问：

python main.py --enable-cors-header

若需局域网其他设备访问，同时加 --listen：

python main.py --enable-cors-header --listen 0.0.0.0

2. 设置服务器地址

打开 生图工具，点击右上角 ⚙️ 按钮，填入 ComfyUI 地址：

  • 本机：http://127.0.0.1:8188
  • 局域网：http://你电脑的IP:8188

3. 选择模型架构

  • SDXL (Checkpoint) — 传统 SDXL 单文件大模型，支持 ControlNet / IP-Adapter
  • Anima (Diffusion Model) — 新架构，扩散模型 + 文本编码器 + VAE 分开加载

4. Anima Base v1.0 模型下载

CircleStone Labs 开源二次元向扩散模型（2B），选 Anima 架构后在三个下拉框分别加载：

  • anima-base-v1.0.safetensors → models/diffusion_models/ — 下载
  • qwen_3_06b_base.safetensors → models/text_encoders/ — 下载
  • qwen_image_vae.safetensors → models/vae/ — 下载

Hugging Face 主页 →

5. SDXL Base 1.0 模型下载

Stability AI 官方 SDXL 基础模型，选 SDXL 架构后在 Checkpoint 下拉框加载：

  • sd_xl_base_1.0.safetensors → models/checkpoints/ — 下载
  • sdxl_vae.safetensors → models/vae/（可选，改善发色） — 下载

6. 局部重绘依赖安装

Anima 局部重绘需要以下 ComfyUI 自定义节点：

  • FLS_Sampler（含 FLS_SamplerV4）
  • ComfyUI-CropAndStitch
  • ComfyUI_essentials
  • Anima LLLite Inpaint 权重

可运行项目提供的 安装局部重绘依赖.bat 一键检查与安装。安装完成后在 生图工具 中即可使用局部重绘。

7. 在线生图（无需本地 ComfyUI）

切换到「在线生图」模式，通过 NAI API 云端出图。适合没有 GPU 或不想配置本地环境的用户。
