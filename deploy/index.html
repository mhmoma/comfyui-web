使用教程

1. 启动 ComfyUI

启动时需要加 --enable-cors-header 参数以允许网页跨域访问：

python main.py --enable-cors-header

如果使用 bat 启动脚本，找到启动文件在 python main.py 后加上该参数。

若需局域网其他设备访问，同时加 --listen：

python main.py --enable-cors-header --listen 0.0.0.0

2. 设置服务器地址

点击右上角 ⚙️ 按钮，填入 ComfyUI 地址：

  • 本机：http://127.0.0.1:8188
  • 局域网：http://你电脑的IP:8188

3. 选择模型架构

左侧面板顶部选择模型架构：

  • SDXL (Checkpoint) — 传统 SDXL 单文件大模型
  • Anima (Diffusion Model) — 新架构，扩散模型 + 文本编码器 + VAE 分开加载

4. 模型资源下载

点击 📥 下载：已连 ComfyUI 且装了 Manager 时自动装到对应文件夹；否则浏览器直接下载，再手动放入 ComfyUI/models/ 下对应目录。

Anima Base v1.0 Anima

CircleStone Labs 开源二次元向扩散模型（2B），擅长动漫插画与概念图。网站选 Anima 架构后，在三个下拉框分别加载下列文件。

anima-base-v1.0.safetensors diffusion_models/
qwen_3_06b_base.safetensors text_encoders/
qwen_image_vae.safetensors vae/
Hugging Face 主页 →
Krea 2 Anima

Krea 第二代开源文生图模型，同样用 Anima 架构加载。Turbo 版出图快，Raw 版质量更高。下列为 Turbo FP8 推荐组合（省显存）；Krea 与 Anima 共用同一 VAE。

krea2_turbo_fp8_scaled.safetensors diffusion_models/
qwen3vl_4b_fp8_scaled.safetensors text_encoders/
krea2_retroanime.safetensors loras/ · 触发词 purple retro anime style
Hugging Face 主页（含 9 款风格 LoRA）→
SDXL Base 1.0 SDXL

Stability AI 官方 SDXL 基础模型，选 SDXL 架构后在 Checkpoint 下拉框加载。适合写实、通用题材，可配合 ControlNet / IP-Adapter。

sd_xl_base_1.0.safetensors checkpoints/
sdxl_vae.safetensors vae/ · 可选，改善发色
Hugging Face 主页 →
IP-Adapter Plus SDXL SDXL 插件

参考图风格迁移，在左侧「风格迁移 IP-Adapter」中启用。需同时下载模型与 CLIP Vision 编码器。

ip-adapter-plus_sdxl_vit-h.safetensors ipadapter/
model.safetensors clip_vision/
ControlNet SDXL SDXL 插件

姿势 / 线稿 / 深度控制，在左侧 ControlNet 面板选用。按需求下载对应模型。

OpenPose XL controlnet/
Canny XL controlnet/

5. 生成图片

  1. 输入正向提示词（可使用标签选择器）
  2. 可选填反向提示词排除不想要的元素
  3. 调整采样参数、图片尺寸
  4. 点击"生成图片"开始

6. 可选功能

  • LoRA — 加载风格微调模型
  • 高清放大 — 生成后自动放大
  • ControlNet — 线稿/深度/姿态控制
  • 参考图 (img2img) — 基于参考图重绘

7. 注意事项

  • HTTPS 页面无法访问 HTTP 的 ComfyUI，建议本地使用
  • Anima 架构暂不支持 ControlNet（官方开发中）
  • 种子设为 -1 表示随机
