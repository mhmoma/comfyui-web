资讯发布流程

文章以 Markdown 管理，存放在 content/news/ 目录：


# 写完 md 后构建
npm run build:news

# 构建 + 同步 deploy
npm run build

构建产物包括 news.json、静态详情页、feed.xml 和 sitemap.xml。
