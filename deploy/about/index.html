关于本站

ComfyUI Web 是什么

ComfyUI Web 是一个面向 AI 绘画爱好者的资讯 + 工具站点。你可以在这里阅读模型更新与教程技巧，也可以直接使用内置的 ComfyUI Web 生图工具——连接本地 ComfyUI 或使用在线 NAI API 出图。

主要功能

  • 资讯站 — 模型动态、教程、工具更新日志
  • 本地生图 — SDXL / Anima 双架构，ControlNet、LoRA、IP-Adapter、高清放大等
  • 在线生图 — NAI API 云端出图，无需本地 GPU
  • 角色库 — 30,000+ 角色 Trigger 与 LoRA 链接
  • 画师库 — 画师标签浏览与自定义画师串
  • 局部重绘 — Anima LLLite 蒙版重绘，多套场景预设
  • 图层合成 / 抠图 — 纯前端图像处理工具

技术栈

  • 前端：纯 HTML / CSS / JavaScript
  • 部署：Cloudflare Pages
  • 数据库：Cloudflare D1（角色/画师数据）
  • 资讯：Markdown + Git 构建（阶段 A）
