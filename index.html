使用教程

1. 启动 ComfyUI

启动时需要加 --enable-cors-header 参数以允许网页跨域访问：

python main.py --enable-cors-header

如果使用 bat 启动脚本，找到启动文件在 python main.py 后加上该参数。

若需局域网其他设备访问，同时加 --listen：

python main.py --enable-cors-header --listen 0.0.0.0

2. 设置服务器地址

点击右上角 ⚙️ 按钮，填入 ComfyUI 地址：

  • 本机：http://127.0.0.1:8188
  • 局域网：http://你电脑的IP:8188

3. 选择模型架构

左侧面板顶部选择模型架构：

  • SDXL (Checkpoint) — 传统 SDXL 大模型
  • Anima (Diffusion Model) — 新架构，需分别选扩散模型/文本编码器/VAE

4. 生成图片

  1. 输入正向提示词（可使用标签选择器）
  2. 可选填反向提示词排除不想要的元素
  3. 调整采样参数、图片尺寸
  4. 点击"生成图片"开始

5. 可选功能

  • LoRA — 加载风格微调模型
  • 高清放大 — 生成后自动放大
  • ControlNet — 线稿/深度/姿态控制
  • 参考图 (img2img) — 基于参考图重绘

6. 注意事项

  • HTTPS 页面无法访问 HTTP 的 ComfyUI，建议本地使用
  • Anima 架构暂不支持 ControlNet（官方开发中）
  • 种子设为 -1 表示随机
